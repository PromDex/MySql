"Представления (Views) в базах данных представляют собой виртуальные таблицы, которые основаны на результатах выполнения запросов к одной или нескольким реальным таблицам в базе данных.
Представления позволяют абстрагироваться от сложных запросов к базе данных и предоставляют удобный интерфейс для доступа к данным. Вот пример создания представления для вашей базы данных:"

-- Создание представления для получения информации о заказах с указанием имени пользователя и названия товара

CREATE VIEW order_details AS
SELECT
    o.id AS order_id,
    u.name AS user_name,
    p.name AS product_name,
    op.total AS quantity
FROM
    orders o
JOIN
    users u ON o.user_id = u.id
JOIN
    orders_products op ON o.id = op.order_id
JOIN
    products p ON op.product_id = p.id;

"Это представление order_details будет возвращать информацию о заказах, включая ID заказа, имя пользователя, название товара и количество товаров в каждом заказе.

Можно создавать представления для различных запросов, которые часто выполняются в системе, чтобы облегчить доступ к данным и упростить разработку запросов."
